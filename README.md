# Pesan-Menu
Aplikasi web berbasis JavaScript yang menampilkan sistem pilihan menu interaktif menggunakan manipulasi DOM. Fitur utama meliputi event listener, perubahan class secara dinamis, serta pengaturan tampilan konten berdasarkan pilihan pengguna. Struktur dibuat menggunakan HTML, dengan styling menggunakan CSS untuk mendukung transisi dan efek visual.
